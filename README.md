This project presents an interactive Job Analytics Dashboard built using Microsoft Excel to analyze employee data and key HR metrics. The dashboard provides meaningful insights into workforce trends, helping in better decision-making.

🔍 Key Features:

Total Employees, Active Employees, and Attrition Count
Attrition Rate & Job Satisfaction Analysis
Gender-wise Employee Distribution
Education-wise Attrition Insights
Department & Job Role Analysis
Age-wise Attrition Trends
Interactive Filters for dynamic data exploration

🛠 Tools Used:

Microsoft Excel
Pivot Tables
Charts & Graphs
Slicers for filtering

💡 Purpose:
The goal of this project is to practice and demonstrate data analysis and visualization skills by converting raw HR data into an easy-to-understand dashboard.

📌 Outcome:
This dashboard helps identify patterns in employee attrition and supports HR teams in making data-driven decisions.
